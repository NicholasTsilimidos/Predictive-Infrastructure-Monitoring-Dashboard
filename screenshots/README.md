This folder contains all visual evidence and interface captures used in the project. These screenshots help demonstrate key steps of the setup process, system monitoring, and predictive alerting features in Zabbix.

- Zabbix Dashboard Overview** — Displays the main dashboard showing real-time system performance.
- Host Monitoring View** — CPU, memory, disk, and network usage for each monitored VM.
- Triggers & Alerts** — Example of a warning or critical alert being activated based on thresholds.
- Trend Predictions** — Zabbix forecast graphs showing projected disk or memory exhaustion.
- Email Notifications** — Example alert sent to the configured administrator email.
- Data Flow Diagram** — Visual representation of server → agent → database → dashboard workflow.

These visuals support the Results section of the main README and provide evidence of a functional monitoring platform.
